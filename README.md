# Welcome to My Basecamp 2
***

## Task
The challenge is to build a collaborative project management platform where multiple users can work together in real time. Managing users, projects, file attachments, and threaded discussions within a single unified system is the core problem this project solves.

## Description
MyBasecamp 2 is a full-stack web application built with Express.js and SQLite. Users can register, create their own projects, invite other users as members, upload file attachments, and start discussion threads with messages. An admin account has full control over all projects and users. The solution uses session-based authentication, Sequelize ORM for database management, and a vanilla JavaScript frontend with no frameworks.

## Installation
```bash
git clone 
cd MyBaeCamp2-Qwasar-master
npm install
node index.js
```

A default admin account is created automatically on first run:
- Email: `admin@mail.com`
- Password: `admin123`

## Usage
http://localhost:3000

- Register a new account or log in with the admin credentials
- Create a new project from the Projects tab
- Open a project to add members, upload files, and start discussions
- As admin, manage all users and projects from the admin panel
